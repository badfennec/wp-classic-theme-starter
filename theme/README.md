# BadFennec WP Starter Theme

A lightweight, "blank canvas" **Classic WordPress Starter Theme** designed for modern development workflows.

This is not a ready-to-use visual theme; it is a technical scaffold intended for developers who need total control over the markup and architecture. It does **not** support Full Site Editing (FSE), sticking to the reliable PHP-based classic structure.
